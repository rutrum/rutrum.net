## Command Line Utilities

* [ccase](https://github.com/rutrum/convert-case/tree/master/ccase): a case conversion tool
* [pathmut](https://github.com/rutrum/pathmut): manipulate path strings
* [album-dl](https://github.com/rutrum/album-dl): a tool that will download and tag albums from a youtube link and a wikipedia article

## Websites

* [stringcase.org](https://stringcase.org): documents the different [string cases](@projects/convert-case)
* [TypeSync](https://typesync.rutrum.net): a typing test game that uses song lyrics, but [it's probably broken](@projects/typesync)

## Minecraft

* [Replant Swamp Trees](https://modrinth.com/mod/replant-swamp-trees): a fabric mod to replant swamp trees that couldn't otherwise be grown
* [Less Crafting](https://github.com/rutrum/less-crafting): a datapack that removes duplicate crafting recipes to prioritize different crafting stations

## Games

* [Fool's Paradise](https://wasm4.org/play/fools-paradise): a game I made for the very first wasm4 game jam, a fantasy console that can run in the browser
